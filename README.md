## Installation

Clone the repository:

```bash
git clone <your-repo-url>
cd SQL_Practice
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run a project:

```bash
python 3.Sql_Pandas_Pipeline.py
```
